# Voice-chatbot
Voice enabled chatbot using Python
This is a Python-based chatbot that supports **both voice and text input**.
It can:
- Answer questions using a CSV dataset
- Fetch information from Wikipedia
- Open popular websites using voice/text commands
- Respond using text-to-speech

This project is part of my learning journey in **AI, NLP, and Python automation**.
